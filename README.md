# Homestead Survival

A 3D farming/survival game built with React, Three.js (React Three Fiber), and Zustand.

## How to Play

1. **Resources**: You start with Wood and Food.
2. **Build**: Click on building buttons at the bottom to select a building.
   - **Cabin**: Basic shelter (Low cost)
   - **Farm**: Generates Food (Cost: Wood, Stone)
   - **Lumber Mill**: Generates Wood (Cost: Wood, Stone)
   - **Mine**: Generates Stone and Iron (Cost: Wood, Stone)
3. **Placement**: After selecting a building, move your mouse (or drag finger) on the ground to choose a spot. Click/Tap to place it.
4. **Economy**: Buildings generate resources automatically over time.
5. **Goal**: Expand your homestead!

## Controls

- **Desktop**:
  - Left Click + Drag to rotate camera
  - Right Click + Drag to pan
  - Scroll to zoom
  - Click to place buildings
- **Mobile**:
  - One finger drag to rotate
  - Two finger drag to pan
  - Pinch to zoom
  - Tap to place buildings

## Tech Stack

- React
- Vite
- React Three Fiber (Three.js)
- Zustand (State Management)
- Tailwind CSS
- React Three Postprocessing
